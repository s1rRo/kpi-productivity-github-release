# Assets Directory

This directory contains application assets like icons, splash screens, and images.

## Required Assets

The following assets are required by Expo:

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen (1284x2778 for iOS)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

## Placeholder Assets

Currently, placeholder assets are used. Replace them with your actual app branding.

You can use online tools to generate assets:
- [Expo Asset Generator](https://www.appicon.co/)
- [MakeAppIcon](https://makeappicon.com/)

## Asset Guidelines

### Icon (icon.png)
- Size: 1024x1024 px
- Format: PNG
- No transparency
- Square shape

### Splash Screen (splash.png)
- Size: 1284x2778 px (iOS max)
- Format: PNG
- Can have transparency
- Centered logo/image

### Adaptive Icon (adaptive-icon.png)
- Size: 1024x1024 px
- Format: PNG
- For Android only
- Background color set in app.json

### Favicon (favicon.png)
- Size: 48x48 px
- Format: PNG
- For web version
